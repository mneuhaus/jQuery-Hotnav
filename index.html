jQuery Key Navigation

This plugin provides a quick and easy way to add a Keyboard Navigation to a Website.

Here is how it works:

Just hit your Alt key and watch the Accesskeys to appear on this Page.
To trigger on of the Items Just press Alt + Accesskey.

Usage:

Download:

jquery.hotnav.zip

Quick and Easy:

<script src="jquery-1.3.2.min.js" type="text/javascript"></script>
<script src="jquery.hotnav.min.js" type="text/javascript"></script>
<script src="jquery.hotkeys.min.js" type="text/javascript"></script>
<script type="text/javascript">
	jQuery("#navigation a").hotnav();
</script>

By default the hotnav runs through all found elements in the order specified and adds an Accesskey to them. Each character will only be used once.

Doing something different:

This will create a Accesskey for the input Field below and focus it on triggering. 
jQuery("#content input").hotnav({
	// Function that gets executed when the Shortcut is triggered
	trigger:function(e){
		e.focus();
	},
	// Function that returns the text to use to determine the Accesskey
	keysource:function(e){
		return e.attr("name");
	},
	// Function 
	insertkey:function(e,key,options){
		e.wrap("<span>").after("<span class='"+options.class+"'>"+key.toUpperCase()+"</span>");
	}
});
</script>

What about regular Input from the User?

The Key Navigation is inactive while an Input Element is focus to enable the User to prevent Compatability Issues.

Default Options:

{
	trigger:function(e,options){
		document.location.href = jQuery(e[0]).attr("href"); 
	},
	keysource:function(e,options){
		return e.text();
	},
	insertkey:function(e,key,options){
		e.append(""+key.toUpperCase()+"");
	},
	key:'alt', // alt, shift or ctrl
	disableInInput:true,
	class:'hotnav-accesskey'
}
